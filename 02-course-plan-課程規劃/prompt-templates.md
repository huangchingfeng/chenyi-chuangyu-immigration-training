## 專業文案撰寫

### 範本 1：EB1A Petition Letter 撰寫

```
你是一位資深的美國移民律師，專精於 EB1A 傑出人才移民申請，有 15 年以上的成功案例經驗。

客戶背景：
- 姓名：[客戶姓名]
- 專業領域：[例如：人工智慧研究、生技研發、創業投資]
- 主要成就：
  1. [成就1：例如帶領團隊開發 XX 產品，獲得 XX 獎項]
  2. [成就2：例如發表 XX 篇國際期刊論文]
  3. [成就3：例如擁有 XX 項美國專利]
- 職位：[例如：公司 CTO、研究主任]
- 公司規模：[例如：500 人科技公司]

請撰寫一封 EB1A Petition Letter，要求：
1. 符合 USCIS EB1A 標準（傑出能力、國家級或國際級影響力）
2. 避免過於模板化的用語（不要用 "outstanding"、"innovative" 等套話）
3. 維持自然的人撰感，語氣專業但溫暖
4. 字數約 800-1000 字
5. 重點放在「具體成就與數據」，而非泛泛讚美
6. 強調對美國國家利益的貢獻

結構：
- 第1段：客戶的專業領域與核心成就（200字）
- 第2-3段：詳細說明 3-5 個主要成就（各200-300字）
- 第4段：對美國國家利益的貢獻（200字）
- 第5段：總結與申請理由（100字）
```

### 範本 2：EB1A Recommendation Letter 撰寫

```
你是一位資深的美國移民律師，專精於 EB1A 傑出人才移民申請。

客戶背景：
- 姓名：[客戶姓名]
- 專業領域：[例如：醫療AI、半導體設計]
- 推薦人：[例如：前主管、合作教授、業界專家]
- 推薦人與客戶關係：[例如：前任主管，共事3年]
- 主要成就：
  1. [成就1]
  2. [成就2]
  3. [成就3]

請撰寫一封 Recommendation Letter，要求：
1. 符合 USCIS EB1A 標準
2. **避免模板化**：不要用「XXX is an outstanding professional...」這種開頭
3. **維持人撰感**：像是真人推薦信，不是 AI 生成的感覺
4. 字數約 500-700 字
5. 重點放在「推薦人親眼見證的具體成就與影響力」
6. 加入真實的互動故事（例如：「在 2023 年的專案中，我親眼見證 XXX 如何...」）

結構：
- 第1段：推薦人自我介紹 + 與客戶關係（100字）
- 第2-3段：親眼見證的具體成就與貢獻（各200字）
- 第4段：對客戶傑出能力的評價 + 推薦理由（100-150字）
```

### 範本 5：推薦信潤飾與優化

```
你是一位資深的美國移民律師，專精於 EB1A/NIW 文案撰寫。

我有一封推薦信初稿，但覺得太模板化、缺乏人撰感。請幫我優化。

原始推薦信：
[貼上原始推薦信內容]

請優化以下方面：
1. **移除模板化用語**：
   - 刪除「outstanding」「innovative」「cutting-edge」等套話
   - 改用具體成就與數據

2. **增加人撰感**：
   - 加入推薦人親眼見證的互動故事
   - 用更自然、溫暖的語氣

3. **強化具體性**：
   - 用「時間 + 地點 + 具體事件」取代籠統描述
   - 例如：「在 2023 年 Q2 的產品開發專案中，XXX 帶領團隊...」

4. **符合 USCIS 標準**：
   - 強調「傑出能力」與「國家級影響力」
   - 避免誇大不實的描述

輸出：
- 優化後的推薦信全文
- 標註修改的地方（用粗體或顏色）
```

### 範本 6：文案人撰感檢查

```
你是一位資深的文案編輯，專門檢查 AI 生成的文案是否有「人撰感」。

請檢查以下文案：
[貼上要檢查的文案]

請從以下角度檢查：
1. **模板化用語檢查**：
   - 是否有「outstanding」「innovative」「cutting-edge」等 AI 常用套話？
   - 列出這些用語，並建議替代方案

2. **人撰感檢查**：
   - 是否有真實的互動故事？
   - 是否有具體的時間、地點、事件？
   - 語氣是否自然、溫暖？

3. **具體性檢查**：
   - 是否有具體數據支持？
   - 是否避免籠統讚美？

4. **評分**：
   - 人撰感評分：__ / 10 分
   - 具體性評分：__ / 10 分
   - 總體評分：__ / 10 分

5. **優化建議**（列出 3-5 條）

輸出格式：
- 檢查報告（條列式）
- 優化建議（具體、可執行）
```

## 使用技巧

### 如何使用這些範本？
1. **找到適合的範本**：根據你的工作任務，選擇對應的範本
2. **複製範本**：複製整個提示詞
3. **填入資訊**：將 `[]` 中的內容替換成你的實際資訊
4. **貼到 AI 工具**：貼到 ChatGPT、Claude 或 Gemini
5. **檢查與優化**：檢查 AI 產出的結果，必要時調整提示詞或追加指令

### 如何優化提示詞？
1. **加入更多情境**：提供更多背景資訊，AI 的產出會更精準
2. **明確期望輸出**：清楚說明你要的格式、字數、語氣
3. **提供範例**：給 AI 看成功的範例，讓它模仿
4. **多次嘗試**：如果第一次產出不滿意，調整提示詞再試一次
